This is an enhanced command-based live coding engine. Press the button to re-evaluate your code.

Features:

  • Specify a rhythmic pattern of beats by entering a space-separated list of times and pitches.
  • Include JavaScript computation in the values.
  • Use square brackets to define repeating sections.
  • Specify the oscillator type at the beginning of the code string.

Example: sawtooth 1+1@220 2@220*2 2[1*3@270]
